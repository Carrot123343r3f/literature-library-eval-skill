# 工程文献库审计报告示例

**问题**：工业视觉缺陷检测在跨产线部署中的方法与性能。  
**类型**：范围综述；**profile**：计算机与 AI；**范围**：2020–2026、同行评议与预印本版本族。

| 维度 | 结果 | 证据状态 | 解释与下一步 |
| --- | --- | --- | --- |
| A1 基准集召回 | 42/48 = 87.5% | 实测 | 48 篇来自两份方法综述与标准数据集论文的 DOI 基准集；6 项进入补检索队列。 |
| A2 查询灵敏度 | 18/20 = 90% | 实测 | 查询快照逐条匹配 gold set；2 项因旧术语未命中，下一轮加入同义词。 |
| A3 候选覆盖 | ≥ 1,620 条去重候选 | 估计下界 | OpenAlex、DBLP 与 IEEE 快照去重；不声称 Recall。 |
| B 检索趋稳 | 未证明稳定 | 实测过程记录 | 后向引文路径尚未完成，不能声称饱和。 |
| C 关键层 | 小样本产线与低照度场景缺口 | 自动初筛，待核验 | taxonomy 显示实验室公开数据集占主导；需人工确认是否属真实证据空白。 |
| D 工程适用性 | 生产线工况字段缺失较多 | 自动初筛 | 多数记录未报告相机、速度与误检成本；不能据此判断方法优劣。 |
| E 更新 | IEEE 最后成功检索距今 3 天 | 实测 | 各来源日期分别列在运行日志。 |
| F 库健康 | DOI 94%，版本族 37 个待核验 | 实测/待核验 | 题录重复与预印本–正式版关系分开记录。 |

**综合判断**：该库可支持范围综述的初步映射，但尚不能宣称检索在声明范围内趋稳。最优先行动是完成后向引文路径、补齐旧术语检索，并人工核验低照度与小样本产线层的版本和相关性。
