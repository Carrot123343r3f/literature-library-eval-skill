# 文献库性质评估模型（B–F 入口）

六个平级维度：A 覆盖（已知文献找回）、B 饱和度（检索是否趋于饱和）、C 平衡（来源与主题是否偏斜）、D 时效（库是否反映当前状态）、E 学术影响/渠道信号（仅背景信号）、F 可用性（文献是否可写综述）。真正的研究质量评估应使用与研究设计匹配的批判性评价工具。

子项权威定义见 [indicator-dictionary.md](indicator-dictionary.md)，默认阈值见 [engineering-standards.md](engineering-standards.md)。不得输出总分；h-core、Tier-1、来源集中度和预印本都只是特定性质的信号，不等同于综述质量。
