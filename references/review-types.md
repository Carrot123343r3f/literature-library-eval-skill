# 综述类型详解（references）

> 5 种支持类型 + 其他类型套用规则。每类：目的/必达/放宽/独特/依据。

## 五种支持类型

### 系统综述（systematic review）

- **目的**：回答**一个具体问题**（如"3DGS-SLAM 哪个精度高"）
- **必达维度**：M1完整PRISMA（四步计数）、A1全域、B饱和、F3去重、E时效
- **放宽**：无（全严）
- **独特**：双人独立筛选、预注册协议
- **依据**：PRISMA 2020 / Cochrane Handbook
- **库标准**：穷尽、无偏、可重复、最新

### 范围综述（scoping review）

- **目的**：**绘制主题全貌/边界**（如"3DGS 都用在哪"）
- **必达**：M1 PRISMA-ScR、A1全域、B饱和、C均衡
- **放宽**：**D质量**（不评证据质量）、E时效
- **独特**：广覆盖但不评判好坏
- **依据**：Arksey & O'Malley 2005 / JBI 2015
- **库标准**：广覆盖各子方向

### 叙事综述（narrative review）

- **目的**：**讲清脉络 + 给洞察**
- **必达**：M1检索式透明、A1 benchmark、**D质量**
- **放宽**：A1全域（不穷尽）、B饱和、E时效、完整PRISMA
- **独特**：代表性+高质量，不要求穷尽
- **依据**：Greenhalgh 等关于 narrative review
- **库标准**：核心抓全 + 引用高质量（CS/AI 多数 survey 属此）

### 快速综述（rapid review）

- **目的**：**紧急加速版系统综述**（时效优先，如政策/疫情决策）
- **必达**：M1简化PRISMA、A1（**时间窗内**全域）、**E1时效必达**、**局限性声明**
- **放宽**：A1全域（时间窗外，不算遗漏）、B（简化筛选）
- **独特**：时间窗限定（刻意限近期）、简化流程（单审/限检索）、必含局限声明
- **依据**：Tricco 等快速综述方法指南
- **库标准**：时间窗内穷尽 + 时效新 + 声明局限

### 伞式综述（umbrella review）

- **目的**：**综述的综述**（汇总已有综述，比较结论/解决矛盾）
- **必达**：M1、A1**综述覆盖**（该主题所有综述都收了吗）、**综述质量（AMSTAR-2/ROBIS）**
- **放宽**：—
- **独特**：
  - **库内是综述论文**（survey/review），非原始研究
  - **AMSTAR-2**（逐综述评 16 项方法学质量）/ **ROBIS**（评综述偏倚）
- **依据**：AMSTAR-2 / ROBIS
- **库标准**：综述全覆盖 + 综述质量达标
- ⚠️ AMSTAR-2 评分超出本 skill 自动范围（手动/专用工具）

## 其他类型（套用规则）

| 类型 | 套用 | 说明 |
| --- | --- | --- |
| 批判性综述（critical） | → 叙事 | 主观批判，不穷尽；报告注明"批判性，主观性强" |
| 元分析（meta-analysis） | → 系统 | 效应量汇总，常与系统结合；需额外统计（I²、Egger）超出本 skill |
| 持续更新综述（living） | → 系统 | 持续版系统综述；时效要求更高 |
| 网络荟萃（network meta） | → 系统 | 多干预比较；需额外统计 |
| 实况综述（realist） | → 叙事 | 理论驱动解释"什么对谁如何起效" |

## 类型判定逻辑（AI 推断 + 用户确认）

1. **看领域**：CS/AI/工程多数 survey=叙事；医学临床决策=系统；新兴领域摸底=范围
2. **看目的**：
   - 回答"哪个更好/更准"→ 系统
   - 了解"领域有什么/边界"→ 范围
   - 讲清"领域怎么发展"→ 叙事
   - 紧急要结论 → 快速
   - 综述太多要汇总 → 伞式
3. **输出推断**：「你的库是[领域]，[目的]→推断[类型]（理由）。确认？」

## 为什么标准按类型不同（核心理由）

- **系统**要可重复+最小偏倚 → 穷尽是核心（A1全域必达）
- **范围**要广覆盖绘全貌 → 不评质量（D放宽）
- **叙事**要代表性+洞察 → 不要求穷尽（A1全域放宽，D必达）
- **快速**要时效 → 时间窗内穷尽（E必达，时间窗外放宽）
- **伞式**对象是综述 → 综述覆盖+综述质量（独特）
- **本质**：穷尽只是系统/伞式的要求，不是所有综述的目的
