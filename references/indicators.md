# A–F 指标总表

| 维度 | 回答的问题 | 子项 |
| --- | --- | --- |
| A 覆盖验证 | 是否命中已知目标与候选边界？ | A1–A4 |
| B 范围结构 | 工程问题的关键切面是否被结构化覆盖？ | B1–B5 |
| C 工程证据适用性 | 证据能否支持工况、基线、指标和标准下的结论？ | C1–C6 |
| D 时效与技术演化 | 检索和版本是否足够新，是否覆盖前沿窗口？ | D1–D5 |
| E 学术影响与知识关联 | 是否连接关键工作、影响力上下文和知识网络？ | E1–E5 |
| F 过程规范、库健康与可复现性 | 检索、筛选、去重、版本和更正能否追溯与复跑？ | F1–F8 |

所有维度平级，不给加权总分。具体量化指标、默认阈值、证据状态和定位动作见 `noncoverage-model.md`；项目覆盖值见 `engineering-standards.md`。
