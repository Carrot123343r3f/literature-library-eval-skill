# 指标详解（references）

> 六维指标 A–F + 快速/伞式独特指标。每项：定义/公式/阈值/依据/数据源/局限/降级。

> **版本说明**：A 维度以 `coverage-model.md` 为准；B–F 的现行判定模型以 `noncoverage-model.md` 为准。本文件中遗留的 GGR/DRR、CV/Gini、h-core、Tier-1、Recency 与 PDF 比率只可作为可选描述性字段，不能单独触发合格/不合格结论。
>
> **阈值说明**：阈值多为方法学经验值（非绝对真理），给**区间**比单值诚实。报告时可按领域/库规模微调，并说明用了哪个阈值。

---

## A 覆盖度

### A1 基准集召回（库的实测覆盖）

- **定义/公式**：`|库 ∩ 基准集| / |基准集|`。匹配优先 DOI、PMID、PMCID、arXiv ID 等稳定标识符；标题相似只能生成待人工核验项。
- **基准集**：由多篇权威综述/指南、领域专家或预先确认的必纳入研究组成；记录来源、版本、纳入规则和构建日期。
- **阈值**：由综述类型与领域 profile 定义。不得将 80% 表述为 Cochrane 或所有领域的统一下限。
- **状态**：有明确基准集且可稳定匹配时为“实测”；无基准集时为“不可评估”。
- **禁止**：数据库查询 count、库总条数、或二者相除均不是 Recall。

### A2 检索策略灵敏度（gold-set validation）

- **公式**：`|检索结果 ∩ gold set| / |gold set|`。它评价检索式，不直接评价当前库。
- **gold set**：10–20 篇或按领域约定数量的已知相关工作；不得与用于事后调词的同一集合混淆。保留独立验证集，或标注“开发集验证”。
- **状态**：保留查询快照和逐条命中表时为“实测”；否则“待人工核验”。

### A3 候选集覆盖估计（非 Recall）

- **定义**：在已声明的数据库、查询、日期和纳入标准下，对候选集覆盖范围的区间估计。
- **方法**：多源去重并集下界、查询灵敏度分析、满足独立性假设时的 capture–recapture。
- **报告**：给区间、数据源、去重规则、独立性风险与检索日期；标为“估计”。
- **禁止**：不得用“库总条数 / 单库命中数”命名为 Recall、DCI 或覆盖率，也不得据此单独判定库不合格。

### A4 核心工作命中

- **定义**：库含公认里程碑的比例
- **公式**：`库∩seminal / seminal集`
- **阈值**：>95%🟢 / 80–95🟡 / <80🔴
- **依据**：领域共识度（共引分析，Small 1973）；seminal 集客观化见 data-sources.md（共引交集）
- **局限**：无官方 seminal 目录；主观性（用共引交集可降）

## B 饱和度

### B1 滚雪球增益 GGR

- **公式**：`第n轮unique新增 / 累计库`
- **阈值**：末轮 <5%🟢 / 5–10🟡 / >10🔴
- **依据**：滚雪球法（Greenhalgh 2005）；饱和概念（Glaser & Strauss 扎根理论，<5% 为定性 saturation 经验阈值）
- **局限**：只对已用方法有效；换源可能再冒新（见 B3）

### B2 重复率 DRR

- **公式**：`本轮已入库 / 本轮总数`
- **阈值**：>70%🟢 / 30–70🟡 / <30🔴
- **依据**：PRISMA 去重步骤（Page 2021）；>70% 表该路径大部分是重复（饱和信号）
- 与 B1 互补印证

### B3 边际新源收益

- **公式**：`新源unique / 新源总数`
- **作用**：多源均<5% 才算真饱和（单方法饱和≠全域饱和）
- **依据**：多源三角验证（triangulation）

## C 均衡

### C1 均衡度（CV + Gini）

- **CV**：`σ(各方向篇数) / μ`
- **Gini**：`Σ(2i−n−1)xᵢ / (nΣxᵢ)`，xᵢ升序
- **阈值**：CV <0.4🟢 / 0.4–0.6🟡 / >0.6🔴；Gini <0.25🟢 / 0.25–0.4🟡 / >0.4🔴
- **依据**：CV/Gini 为统计学经典分布度量；<0.4/<0.25 为"中等变异"惯例（无绝对标准，领域依赖）
- **局限**：均衡≠全（可能都浅）；领域天然不均（热门方向大）是真实，非分类错——CV 高时先判"领域不均"还是"搜集偏科"

### C2 来源多样性

- **Shannon**：`H=−Σpᵢln(pᵢ)`，归一化 `J=H/ln(n)`
- **阈值**：J >0.6🟢 / 0.4–0.6🟡 / <0.4🔴
- **依据**：Shannon 1948（信息熵）；J 为均匀度（Pielou），>0.6 为"较均匀"生态学经验
- pᵢ = 来源i篇数/总

### C3 各方向 Recall

- **公式**：`方向k库 / 方向k全域`
- **阈值**：>40%🟢 / 20–40🟡 / <20🔴
- **依据**：A1 Recall 的方向化；发现真薄弱（篇数会误判）
- **注意**：全域按方向（OpenAlex cites+方向词 count）

## D 质量

### D1 h-core

- **定义**：`max{h: 至少h篇被引≥h}`
- **依据**：Hirsch 2005（h-index）
- **判据**：相对比较（同领域库间），**无绝对阈值**（h 随领域/库规模变）
- **数据源**：OpenAlex `cited_by_count`（按**标题**搜取主记录/合并版被引；按 arXiv DOI 查低估）
- **局限**：偏老论文（被引累积）；按 arXiv 版查是下界

### D2 Tier-1 占比

- **公式**：`顶会顶刊 / 总`
- **阈值**：无绝对（相对比较）；叙事关注、范围放宽
- **Tier-1 定义**（注明用哪个）：CCF-A+一区 / 或领域顶会列表（见 domains.md）
- **依据**：同行评议作为质量代理（ Garfield 的引文索引思想）
- **局限**：质量代理≠质量本身

## E 时效

### E1 Recency

- **公式**：`近N年 / 总`（爆发领域 N=2，成熟 N=5）
- **阈值（爆发）**：>40%🟢 / 25–40🟡 / <25🔴
- **依据**：领域惯例（爆发领域近2年应占多数）；快速综述 E1 提**必达**（时效是核心价值）
- **N 选择**：按领域成熟度（爆发=2，成熟=5），报告标注用了哪个 N

### E2 前沿覆盖

- **公式**：`最新年库 / 该年全域`
- **作用**：Recency 看库内占比，E2 看最新年绝对覆盖，互补

### E3 预印本占比（信息性指标，不标红黄绿）

- **定义**：库内来自预印本平台（arXiv / bioRxiv / medRxiv 等）的论文 ÷ 总数
- **公式**：`预印本来源篇数 / 总`（venue 或 url 含 arXiv/bioRxiv/medRxiv 计为预印本）
- **含义（双刃剑）**：
  - ✅ 高 = **前沿性**（预印本先于正式发表 1–2 年）+ **开放性**（免费可读）
  - ⚠️ 但 = **未经同行评议**，质量未验证；正式版可能有重大修改或撤稿
- **领域依赖**：CS/AI 占比高属正常（arXiv 文化强）；医学/社科偏低（更重同行评议）。**无绝对阈值**。
- **判据**：**不标红黄绿**（双刃剑，不评高低）；报告给值 + 解读（如"51% 预印本，CS 领域惯例，前沿性好，但需注意未评议风险，关键结论以正式版为准"）
- **依据**：预印本作为最新研究进展的代理（arXiv 模式，Ginsparg；bioRxiv/medRxiv 同理）

## F 可用

### F1 摘要率

- **公式**：`有摘要 / 总`，阈值 >90%🟢 / 70–90🟡 / <70🔴
- **依据**：摘要完整性影响筛选可行性（<70% 难做 title/abstract screening）

### F2 PDF 率

- **公式**：`有全文 / 总`，阈值 >70%🟢 / 50–70🟡 / <50🔴
- **依据**：全文可得影响深度阅读/数据提取；>70% 为实用经验
- **降级**：顶会正式版无免费 PDF→arXiv 预印本替代（内容等同）

### F3 去重干净

- **公式**：`残留重复 / 总`，阈值 <2%🟢 / 2–5🟡 / >5🔴
- **依据**：PRISMA 去重质量（Page 2021）；>5% 影响结论（重复计数）

## 独特指标（特定类型）

### 快速综述：时间窗覆盖

- **定义**：库限定时间窗（如近3年）内的覆盖
- **A1 改为时间窗内全域**（时间窗外不算遗漏——刻意设计）
- **局限性声明**：报告必须含"因时间限制，检索/筛选简化，可能有遗漏"
- **依据**：快速综述方法指南（Tricco 等）

### 伞式综述：综述覆盖 + 综述质量

- **综述覆盖**：`库内综述数 / 该主题所有综述`（A1 分母改为综述集）
- **综述质量（AMSTAR-2）**：逐篇综述评 16 项方法学质量（Shea 2017）
  - ⚠️ 复杂，**超出本 skill 自动范围**——需手动或专用工具
  - skill 能算综述覆盖度，AMSTAR-2 标注"待手动评"
- **ROBIS**（Whiting 2016）：评综述偏倚风险（AMSTAR-2 替代）
- **依据**：AMSTAR-2 / ROBIS（综述方法学质量评估工具）

## 阈值总说明（重要）

- 阈值是**经验值**（引方法学惯例），非绝对真理。
- 报告时**说明用了哪个阈值**，允许按领域/库规模微调。
- 关键是**红黄绿 + 阻断项**的判据逻辑（按类型），比绝对数值更重要。
- 边界值（如 79% vs 80%）不要过度解读——给区间更诚实。
