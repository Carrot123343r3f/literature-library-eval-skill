# 工程数据源与证据等级

优先按 `engineering-profiles.md` 为工程问题选择来源。开放源用于可重复回退，不等价于专业订阅索引。

| 用途 | 最低可接受证据 | 不能替代它的东西 |
| --- | --- | --- |
| A1/A2 匹配 | 有来源和日期的 DOI、PMID、PMCID、arXiv 或 OpenAlex ID 集合 | 标题相似、库条数、引用量 |
| A3 下界 | 两个以上来源的去重候选并集和纳入边界 | 单一数据库结果数 |
| A3 区间 | 满足独立性、可靠去重与稳定边界的捕获–重捕获假设 | 未验证的数据库重叠比 |
| 工程适用性 | 任务、工况、基线、指标、标准或数据集字段 | venue 或被引排名 |
| 更正状态 | 有日期的权威来源检查或人工核验 | 输入题录中缺失标志 |

每个来源记录检索式、字段、过滤器、日期、返回量、分页/截断情况、失败原因和去重规则。只有取得合法访问权限时才能查询订阅源。
